This property is used to create link to logs created during build run so testers can look into them for more detailed info. To customize logs location logs location can be customized. Pattern can contain up to 2 placeholders.
Delete white space between $ and {} in placeholders.

$ {projectName}
project name, which is build part of
$ {sequentialNumber}
of executed build
Example: http://<YOUR_DOMAIN>/build/$ {projectName}/$ {sequentialNumber}/logs For executed build run with YOUR_DOMAIN=mydomanin.cz, projectName=myProject and sequentialNumber=5 Will create link eq: http://mydomanin.cz/build/myProject/5/logs
